# Aegis-Linux

# Aegis-Linux é uma solução de controle parental de nível sistêmico para distribuições Linux, projetada para ser gerenciada 
# inteiramente de forma remota através de uma interface web hospedada no GitHub.

Diferente de controles locais tradicionais, o Aegis-Linux utiliza o GitHub como um "servidor de comando e controle" (C2), permitindo que administradores gerenciem sessões, definam limites de tempo e apliquem bloqueios imediatos de qualquer lugar do mundo
